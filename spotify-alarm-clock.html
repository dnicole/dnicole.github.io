Use Spotify As Your Alarm Clock

Here's a little script to use Spotify as an alarm clock. It will turn up your system volume to 100%, open Spotify to the playlist you specify, and shuffle to start playing a random track.

If you can get this to run using the Windows Task Scheduler on Windows 10, please let me know. That said, I've set it to successfully run using System Scheduler.
